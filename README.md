# adaptations-archaea

In prokaryotes, the evolutionary and environmental factors behind the large variation in genome size is not entirely clear. One common hypothesis is that thermophiles and hyperthermophiles have smaller genomes due to selection for efficient replication or genome stability under such extreme conditions.
